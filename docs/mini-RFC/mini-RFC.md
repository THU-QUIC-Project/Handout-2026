# mini-RFC

In this experiment, we use a simplified version of QUIC (RFC 8999 9000 9001 9002).

Since modification of extracts from IETF RFCs is not granted by IETF as original authors retain their copyrights to RFCs, we can not put the mini-RFCs directly on the website.

Please refer to the course repo for our [mini-RFCs](https://github.com/THU-QUIC-Project/mini-RFC).
